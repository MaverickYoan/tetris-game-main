# Fichiers Audio Requis pour le Jeu Tetris

Placez les fichiers audio suivants dans ce dossier :

## Fichiers Nécessaires

1. **line_clear.wav** - Son joué quand une ou plusieurs lignes sont complétées
   - Durée recommandée : 0.2-0.5 secondes
   - Style : Son joyeux, ascendant

2. **piece_land.wav** - Son joué quand une pièce se pose
   - Durée recommandée : 0.05-0.1 secondes
   - Style : Son court et sec

3. **game_over.wav** - Son joué à la fin de la partie
   - Durée recommandée : 0.5-1 seconde
   - Style : Son descendant, mélancolique

4. **achievement.wav** - Son joué quand un succès est débloqué
   - Durée recommandée : 0.4-0.6 secondes
   - Style : Fanfare joyeuse

5. **hold.wav** - Son joué quand on utilise la fonction Hold/Reserve
   - Durée recommandée : 0.1-0.2 secondes
   - Style : Son de confirmation

## Format Recommandé

- Format : WAV (PCM)
- Fréquence d'échantillonnage : 44100 Hz
- Bits : 16-bit
- Canaux : Mono ou Stéréo
- Taille : < 100 KB par fichier

## Sources de Sons Gratuits

- **Freesound.org** - Bibliothèque de sons gratuits
- **OpenGameArt.org** - Sons pour jeux vidéo
- **Zapsplat.com** - Effets sonores gratuits
- **Sonniss.com** - Packs de sons gratuits

## Note

Si un fichier est manquant, le jeu continuera de fonctionner sans ce son spécifique (pas de crash).
